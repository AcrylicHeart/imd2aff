# 节奏大师(imd)转Arcaea(aff)

适用于arc4.0以后,2024年愚人节之前的版本

## 使用方法

在工作目录中创建两个文件夹：`input` 和 `output`。

在input中放入要转换的imd，imd的文件名称格式要符合以下习惯：

woaini_4k_hd.imd
woaini_5k_ez.imd
...

运行py代码后会在output文件夹中生成对应的aff,如下

woaini_4k_hd.aff
woaini_5k_ez.aff
...
